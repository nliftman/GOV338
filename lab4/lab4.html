Overview

For this lab, you can either use your group’s final project data, or the sample network data provided. The sample data is on the latent network of state policy adoptions on civil rights. If your network is particularly challenging to plot/visualize, I suggest you use the provided data, but that isn’t a requirement.

Part 1- loading data and summary statistics

  1. Load your data, convert it into a network, and plot it. For now, only load rio , igraph, and tidyverse.

  2. Provide the following descriptive of the network: number of edges, number of nodes, density, and report the minimum, maximum, and mean degree in our network.

  3. Identify and list the 3 nodes with the highest degree in the data.

Part 2- Role Analysis

  1. Load concoR and construct a correlation of ties between your nodes. Look at the first column of the correlation matrix, which reflects the correlations between that node and all others in the data. Which node is most strongly correlated with the first node? Hint you need to first extract an adjacency matrix from your igraph object.

  2. Use Concor’s hierarhcical clustering algorithm (concor_hca) to partition you matrix. How many blocks are produced?

  3. Assign the blocks to your nodes in the network and plot the network with node color being determined by block membership.

  4. Look at the plot, the correlation matrix, and your calculations for node degree. Were you surprised by which groups were put in the same block?

Part 3- Dendograms and collapsed block graphs

  1. You can now load SNA and network. Use the equiv.clust command to calculate structurally equivalent nodes in your matrix. Plot the results.

  2. Partition your data (rect.hclust command) by the number of blocks you believe best divide the plot produced in 8. Re-make the plot with the partitions overlaid.

  3. Estimate the blockmodel (blockmodel) with the specified number of divisions you felt was the best fit for 9. Plot the collapsed version of the block model (see gplot sample code).

  4. Lastly, look at the nodes that are assigned to the same blocks. Can you identify any similar between these nodes, either in how they appear in the visualized network, or based on some characteristics of the nodes? For example, are states grouped by ideology/region, etc?
